## 给前端开发者的快速提示

- **变量声明**: 不需要 `var`, `let`, `const`。直接 `x = 10`。
- **代码块**: 不使用 `{}`，而是使用**缩进** (Indentation)。
- **打印**: `console.log` -> `print()`
- **数组**: `Array` -> `List` (`[]`)
- **对象**: `Object` -> `Dictionary` (`{}`)
- **空值**: `null` -> `None`
- **布尔值**: `true/false` -> `True/False` (首字母大写)

## 如何运行代码

确保你已经安装了 Python (建议 3.10+)。

在终端中运行：
```bash
python path/to/file.py
# 或者
python3 path/to/file.py
```
